Breccia Mode for Emacs

Here is package Breccia Mode for Emacs. This package introduces a major mode for editing Breccian text.

If you install this package using a package manager, then already Breccia Mode should activate for any `.brec` file you load. Alternatively, see `breccia-mode.el` for instructions to install it manually.

To view a list of customizeable faces, use `M-x customize-group <RET> breccia <RET>` — your buffer must be in Breccia Mode for this to work. Or look through the `defface` definitions in `breccia-mode.el`. For reference, there is a working example of customization:
