Brec Mode for Emacs

This project implements Brec Mode, an Emacs major mode for editing Breccia.

Here is a screen shot.

Installation

You can install Brec Mode from MELPA using a package manager such as ` package.el`. Or you can install it manually.

See `brec-mode.el` for instructions.
